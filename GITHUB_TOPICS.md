- `download-manager`
- `npm`

- `winget`
- `golang`

- `tui`

## Secondary Topics
- `pip`
- `cargo`
- `maven`
- `dependency-management`
- `offline-installer`
- `mirror`
- `batch-download`
- `windows`
- `cross-platform`

## How to Add Topics

1. Go to your repository on GitHub
2. Click "About" (gear icon) in the right sidebar
3. Add topics in the "Topics" field
4. Click "Save changes"
