# Source Fetcher Web GUI 使用指南

## 🚀 快速启动

### 方法 1: 使用启动脚本（推荐）
```powershell
.\start-gui.ps1
```

### 方法 2: 直接运行
```powershell
.\source-fetcher.exe gui
```

### 方法 3: 使用别名
```powershell
# 如果已安装别名
sfer gui
```

浏览器会自动打开 `http://localhost:8765`

## 📖 功能说明

### 🔍 Search（搜索）标签页

**搜索包：**
1. 选择 Source（来源）：all / npm / choco / winget
2. 输入 Query（关键词）
3. 可选：设置 Mirror（镜像）、Limit（结果数量）
4. 点击 **Search** 按钮

**操作结果：**
- 点击包名查看详细信息
- 勾选复选框选择多个包
- 点击 **Download Selected** 下载选中的包
- 点击 **Install Selected** 安装选中的包
- 点击 **Select All** 全选
- 点击 **Clear** 清空选择

### 📋 Queue（队列）标签页

**查看任务：**
- 显示所有下载和安装任务
- 实时更新任务状态
- 状态图标：
  - ⏳ pending（等待中）
  - 🔄 running（运行中）
  - ✅ completed（已完成）
  - ❌ failed（失败）

**管理队列：**
- 点击 **Refresh** 刷新队列
- 点击 **Clear Finished** 清理已完成的任务

### 🌐 Mirrors（镜像）标签页

**测试镜像速度：**
1. 选择 Source（来源）
2. 点击 **Test Mirrors** 开始测速
3. 查看结果：
   - ✅ 可用的镜像
   - ❌ 不可用的镜像
   - 延迟时间（越小越快）

### ℹ️ About（关于）标签页

查看软件信息、版本号、功能说明等。

## 🎨 界面特性

- **深色主题** - 护眼舒适
- **响应式设计** - 适配不同屏幕
- **实时状态** - 底部状态栏显示操作结果
- **快捷操作** - 鼠标点击即可完成所有操作

## 💡 使用技巧

1. **批量下载**
   - 搜索包后，勾选多个包
   - 点击 Download Selected 一次性加入队列

2. **镜像加速**
   - 先在 Mirrors 标签页测速
   - 找到最快的镜像
   - 在 Search 页面的 Mirror 字段填入镜像 URL

3. **查看详情**
   - 点击搜索结果中的任意包
   - 下方 Details 区域会显示完整信息

4. **队列管理**
   - 任务会自动按顺序执行
   - 可以随时切换到 Queue 标签查看进度
   - 完成后点击 Clear Finished 清理

## 🔧 故障排除

### 浏览器没有自动打开
手动访问：`http://localhost:8765`

### 端口被占用
如果 8765 端口被占用，程序会自动尝试其他端口。

### 无法连接服务器
1. 检查防火墙设置
2. 确认 source-fetcher.exe 正在运行
3. 尝试重启程序

### 搜索/测速失败
1. 检查网络连接
2. 尝试使用镜像
3. 查看状态栏的错误信息

## 🛑 停止服务器

在运行 `source-fetcher.exe gui` 的终端窗口按 `Ctrl+C`

## 📝 注意事项

- Web GUI 仅在本地运行（localhost），不会暴露到外网
- 关闭浏览器标签页不会停止服务器
- 必须在终端按 Ctrl+C 才能停止服务器
- 服务器运行时可以打开多个浏览器标签页访问
