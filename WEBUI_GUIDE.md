# Source Fetcher Web UI 使用指南

## 🚀 启动

```bash
# 使用 sfer 别名
sfer gui

# 或直接运行
.\source-fetcher.exe gui

# 浏览器会自动打开 http://localhost:8765
```

## 📋 功能概览

### 1️⃣ 搜索 (Search)

**搜索包**
- 选择源（npm, pip, cargo, maven, choco, winget, all）
- 输入包名（自动搜索，结果实时更新）
- 可选：指定镜像、限制结果数量

**操作结果**
- 点击选择包
- 双击复制包名
- 批量选择：使用 `Ctrl+A` 或点击"全选"按钮
- 点击结果查看详情
- 使用 `D` 键下载、`I` 键安装

**结果颜色标识**
- 🔴 **npm** - Node.js 包
- 🔵 **pip** - Python 包
- ⚫ **cargo** - Rust 包
- 🟣 **maven** - Java 包
- 🔷 **choco** - Chocolatey 包
- 🟦 **winget** - Windows 包

### 2️⃣ 队列 (Queue)

**查看任务**
- 实时显示所有下载/安装任务
- 任务状态：等待中、运行中、已完成、失败

**管理队列**
- 🔄 刷新：手动更新队列状态（快捷键 `R`）
- 🗑️ 清理：删除已完成和失败的任务

**任务状态**
- ⏳ **等待中** - 任务在队列中
- 🔄 **运行中** - 正在执行（带进度条）
- ✅ **已完成** - 成功完成
- ❌ **失败** - 执行失败（显示错误信息）

### 3️⃣ 镜像 (Mirrors)

**测试镜像速度**
- 选择源
- 点击"测试镜像"
- 查看每个镜像的响应时间

**结果排序**
- 自动按速度排序（最快在前）
- 前三名显示 🏆 TOP 标识
- 快速镜像有绿色边框高亮
- 双击镜像 URL 复制地址

### 4️⃣ 关于 (About)

- 查看版本信息
- 支持的包源列表
- 功能特性介绍
- 键盘快捷键列表
- 作者和许可证信息

## ⌨️ 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + K` | 聚焦搜索框 |
| `1` - `4` | 切换标签页 (搜索/队列/镜像/关于) |
| `Ctrl/Cmd + A` | 全选搜索结果 |
| `D` | 下载选中的包 |
| `I` | 安装选中的包 |
| `R` | 刷新队列 |
| `Esc` | 清除选择 |
| `?` | 显示所有快捷键 |

## 💡 使用技巧

### 快速搜索
- 输入 2 个字符后自动触发搜索
- 无需按回车，结果实时更新
- 搜索结果会自动缓存，相同查询秒出

### 批量操作
1. 搜索包
2. `Ctrl+A` 全选或手动多选
3. 按 `D` 批量下载或 `I` 批量安装
4. 自动跳转到队列页面查看进度

### 复制操作
- 双击搜索结果：复制包名 `package@version`
- 双击镜像 URL：复制镜像地址
- 详情面板：点击复制按钮

### 镜像加速
1. 切换到"镜像"标签
2. 选择源并测试
3. 查看最快的镜像（TOP 1-3）
4. 在搜索时手动指定快速镜像

### 队列管理
- 队列自动每 5 秒刷新
- 任务按顺序执行（串行）
- 可以在任务执行时继续搜索和添加
- 定期清理已完成的任务

## 🐛 故障排除

### 搜索无结果
- 检查网络连接
- 尝试切换镜像
- 检查包名拼写
- 尝试其他源（all）

### 下载/安装失败
- 查看队列中的错误信息
- 检查磁盘空间
- 验证网络连接
- 尝试使用镜像加速

### 页面无响应
- 刷新浏览器页面（F5）
- 检查服务器是否运行
- 查看控制台错误信息

### 端口被占用
- 修改端口号（需要修改代码）
- 或关闭占用 8765 端口的程序

## 🔧 高级功能

### 私有源配置
需要配置 `source-fetcher.yaml` 文件，暂不支持 GUI 配置。

### 批量任务
使用 YAML 配置文件，通过 CLI 执行：
```bash
sfer batch --config source-fetcher.yaml
```

### 断点续传
Web UI 自动支持断点续传，无需配置。

### 并发下载
当前 Web UI 使用默认设置，支持分块下载。
