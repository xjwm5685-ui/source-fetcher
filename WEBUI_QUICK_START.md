# Web UI 快速启动指南

## 🚀 立即开始

### 1️⃣ 编译（必需，代码已更新）

```bash
cd d:\dy\source-fetcher
go build -o source-fetcher.exe
```

### 2️⃣ 启动

```bash
.\source-fetcher.exe gui
```

浏览器会自动打开 `http://localhost:8765`

### 3️⃣ 开始使用

1. **搜索包**
   - 在搜索框输入包名（如 `lodash`）
   - 选择源（npm, pip, cargo, maven, choco, winget, all）
   - 结果实时显示

2. **下载包**
   - 勾选想要的包
   - 点击绿色下载按钮 📥
   - 文件下载到当前目录

3. **安装包**（仅 npm）
   - 搜索 npm 包
   - 勾选想要的包
   - 点击蓝色安装按钮 📦
   - 自动安装到 `node_modules`

4. **查看队列**
   - 点击"Queue"标签页
   - 查看所有任务状态
   - 自动每 5 秒刷新

## ⌨️ 快捷键

- `Ctrl+K` - 聚焦搜索框
- `1-4` - 切换标签页
- `D` - 下载选中的包
- `I` - 安装选中的包
- `?` - 查看所有快捷键

## 🔍 调试

按 `F12` 打开开发者工具，查看控制台日志。

## 📝 常见操作

### 批量下载
```
1. 搜索包
2. Ctrl+A 全选
3. 点击下载按钮
4. 切换到队列查看进度
```

### 测速选择最快镜像
```
1. 切换到"Mirrors"标签
2. 选择源
3. 点击"Test Mirrors"
4. 查看响应时间
5. TOP 1-3 是最快的镜像
```
