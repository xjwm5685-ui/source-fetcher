## [Unreleased] - 2026-06-05

### Web UI 重大更新 🎉

#### ✨ 新增功能

- **实时搜索**: 输入时自动搜索，结果实时更新（800ms 防抖）
- **键盘快捷键**: 完整的键盘导航系统
  - `Ctrl/Cmd + K` 聚焦搜索
  - `1-4` 切换标签页
  - `D`/`I` 下载/安装
  - `R` 刷新队列
  - `?` 查看帮助
- **Toast 通知系统**: 优雅的消息提示（成功/错误/警告/信息）
- **一键复制功能**: 双击结果复制包名，双击镜像复制 URL
- **搜索结果缓存**: 相同查询秒出（最多缓存 50 个）
- **进度指示**: 队列中运行任务显示实时进度条
- **快捷键帮助**: 按 `?` 显示所有快捷键
- **欢迎提示**: 首次打开显示使用提示

#### 🎨 视觉改进

- **包源颜色标识**: 每个包源有独特颜色（npm红、pip蓝等）
- **状态徽章系统**: 清晰的任务状态标识
- **TOP 镜像标识**: 前 3 名镜像显示 TOP 徽章
- **快速镜像高亮**: <500ms 的镜像绿色边框高亮
- **改进的空状态**: 更友好的空状态提示和图标
- **增强的动画效果**: 
  - Toast 滑入/滑出动画
  - 状态变化脉冲动画
  - 空状态浮动动画
  - 复制反馈动画
  - 悬停光晕效果

#### 💪 体验优化

- **增强的队列管理**:
  - 状态徽章显示（等待中/运行中/完成/失败）
  - 任务状态颜色区分
  - 清除队列前确认对话框
  - 详细的错误信息显示

- **改进的镜像测试**:
  - 智能排序（按速度，失败的在后）
  - TOP 标识和速度高亮
  - 双击复制 URL

- **更好的搜索体验**:
  - 实时搜索结果
  - 结果计数正确的复数形式
  - 详细的工具提示
  - 改进的详情面板格式

#### 📱 响应式增强

- **移动端优化**: 改进小屏幕布局和触摸交互
- **多设备支持**: 4 个响应式断点（Desktop/Tablet/Mobile/Small）
- **自适应 Toast**: Toast 通知响应式定位
- **优化的按钮大小**: Touch 友好的交互元素

#### ♿ 可访问性

- **完整的键盘支持**: 所有功能都可通过键盘操作
- **焦点可见性**: 清晰的焦点指示器
- **减少动画模式**: 支持 `prefers-reduced-motion`
- **语义化 HTML**: 更好的屏幕阅读器支持

#### 🐛 Bug 修复

- 修复 CSS `-webkit-line-clamp` 缺少标准属性警告
- 修复搜索结果选择状态同步问题
- 修复响应式布局在小屏幕的显示问题
- 改进错误处理和用户反馈

#### 📚 文档

- 新增 `WEBUI_GUIDE.md` - 完整的 Web UI 使用指南
- 新增 `WEBUI_IMPROVEMENTS.md` - 详细的改进说明
- 新增 `WEBUI_TEST_CHECKLIST.md` - 测试清单
- 更新 About 页面，添加键盘快捷键说明

#### 🔧 技术改进

- **性能优化**: 搜索防抖、结果缓存、请求取消
- **代码质量**: 模块化函数、改进的错误处理、更好的注释
- **浏览器兼容**: Clipboard API fallback、CSS 前缀、标准属性支持
