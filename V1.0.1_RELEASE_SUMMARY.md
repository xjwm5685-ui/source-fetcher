## 📋 GitHub Release 说明模板

复制以下内容到 GitHub Release:

```markdown
# 🚨 Security Patch Release - Upgrade Immediately

## Critical Security Fixes

This release fixes **3 critical security vulnerabilities**:

1. **CORS Vulnerability (CVE-pending)** - Remote attackers could control local instance
2. **Command Injection Risk** - Malicious package names could execute commands
3. **SSRF Vulnerability** - Could access internal network resources

**All users should upgrade immediately.**

## ✨ New Features

- 🎛️ **Configuration File Support** - Global `.source-fetcher.yaml` configuration
- 🏥 **Health Check** - Faster and more reliable Web GUI startup
- ⏹️ **Cancellation Support** - Ctrl+C to interrupt operations

## 🐛 Bug Fixes

- Fixed winget error handling
- Hardened file permissions (0600/0640)
- Added safe YAML parsing

## 📚 Documentation

- Added SECURITY.md with security policy
- Updated README.md with configuration guide
- Added comprehensive code review reports

## 💔 Breaking Changes

None - Fully backward compatible

## ⬆️ Upgrade

1. Download `source-fetcher-v1.0.1-windows-amd64.exe`
2. Replace your existing binary
3. Verify: `source-fetcher version`

See [RELEASE_NOTES_v1.0.1.md](RELEASE_NOTES_v1.0.1.md) for details.

---

**Full Changelog**: https://github.com/xjwm5685-ui/source-fetcher/compare/v1.0.0...v1.0.1
```
