# Web UI 快速参考卡片

## 🚀 启动命令

```bash
go build -o source-fetcher.exe && .\source-fetcher.exe gui
```

## ⌨️ 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+K` | 聚焦搜索框 |
| `1` `2` `3` `4` | 切换标签页 |
| `Ctrl+A` | 全选结果 |
| `D` | 下载选中 |
| `I` | 安装选中 |
| `R` | 刷新队列 |
| `Esc` | 清除选择 |
| `?` | 显示帮助 |

## 🎯 常用操作

### 搜索并下载
```
1. 输入包名 (如: lodash)
2. 选择源 (如: npm)
3. 勾选结果
4. 按 D 或点击下载按钮
```

### 批量操作
```
1. 搜索
2. Ctrl+A 全选
3. 按 D 下载
```

### 安装 npm 包
```
1. 搜索 npm 包
2. 勾选
3. 按 I 或点击安装按钮
4. 切换到队列查看进度
```

### 测试镜像速度
```
1. 切换到 Mirrors 标签
2. 选择源
3. 点击 Test Mirrors
4. 查看 TOP 1-3
```

## 💡 交互技巧

- **双击结果** → 复制包名
- **双击镜像 URL** → 复制地址
- **点击下拉框** → 展开选项
- **↑↓ 键** → 选择下拉选项
- **Enter** → 确认选择
- **Esc** → 关闭下拉框

## 🐛 调试

按 `F12` 打开开发者工具查看：
- 请求日志
- 响应数据
- 错误信息
- 队列状态

## 📍 文件位置

- 下载文件: 当前目录
- 安装目录: `node_modules`
- 缓存目录: `.source-fetcher`

## 🆘 快速帮助

**搜索无结果？**
- 检查网络
- 尝试其他镜像
- 检查包名拼写

**下载失败？**
- 查看队列错误信息
- 检查磁盘空间
- 查看服务器日志

**安装失败？**
- 确认是 npm 包
- 包名和版本存在
- 查看详细错误
