About

A unified package manager for downloading and installing packages from multiple sources with a beautiful, modern interface.

Supported Sources

  • npm — Node.js packages
  • Chocolatey — Windows packages
  • Winget — Windows Package Manager

Features

  • Search packages across multiple sources
  • Download with resume support
  • Install with dependency resolution
  • Mirror support for faster downloads
  • Batch operations via YAML config
  • Keyboard shortcuts for power users

Keyboard Shortcuts

  • Ctrl+K Focus search
  • 1-4 Switch tabs
  • D Download selected
  • I Install selected
  • ? Show all shortcuts
